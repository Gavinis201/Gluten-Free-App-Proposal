Project Overview

Lowest-cost first launch

The Basic MVP is designed to get the core website online at the lowest project price. It includes the foundation needed for users to create accounts, purchase access, browse gluten-free restaurant/menu information, and for the owner to manage core content from a private admin area.

Included Features

What is included

  • User authentication: account creation, secure sign-in, email verification, and forgotten password reset.
  • Basic role-based permissions: access levels for admins, paid users, and general users.
  • Payment access: subscription or one-time payment processing through Stripe or PayPal.
  • Restaurant and menu pages: core restaurant/menu content with descriptions, gluten-free status, notes, and basic image support.
  • Search and basic filters: users can search by park, restaurant, menu item, and basic dining categories.
  • Admin Content Management: private admin tools for parks, restaurants, menu items, and core website content.
  • Notification email configuration: setup for automated account emails, including verification and password reset messages.
  • Launch setup: GitHub setup, domain setup through GoDaddy or Cloudflare, hosting setup, and production launch.

Admin Content Management

Basic admin tools

  • Park Management: create and edit park records.
  • Restaurant Management: create, edit, archive, and manage restaurant information.
  • Menu Management: create, edit, archive, and organize menu items by restaurant and meal type.
  • User Access Management: manage basic user access and paid-access status.
  • Account Email Settings: configure the sending email used for account verification and password reset emails.
