Platform Clarification

Responsive Website

This proposal includes the design and development of a custom responsive web application that runs in a web browser on desktop, tablet, and mobile devices. It is optimized for mobile use and does not include native iPhone or Android applications unless specifically noted.

Included

  • Responsive website / web application.
  • Works on desktop, tablet, and mobile browsers.
  • Can be saved to a phone home screen for quick access.

Not Included

  • Native iPhone application.
  • Native Android application.
  • Apple App Store or Google Play Store submission.
