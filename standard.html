Project Overview

Recommended Production-Ready Website

The Standard Build package is a complete production-ready responsive web application centered around a custom content management system designed specifically for managing gluten-free dining throughout the Disneyland Resort, including parks, maps, restaurant pins, menu items, reviews, contributor approvals, permissions, and community updates.

Fixed Project Price$8,500
Package TypeStandard Build

Included Features

What Is Included

  • User authentication: account creation, secure sign-in, email verification, and forgotten password reset.
  • User settings page: users can change their password or delete their account.
  • Role-based permissions: access levels for admins, approved contributors, paid users, and general users.
  • Subscription management: paid member access, user permissions, subscription or one-time payment processing, and automated account emails.
  • Custom Content Management System (CMS): complete control over parks, maps, restaurant pins, menu items, reviews, contributor approvals, permissions, and community updates.
  • Map and direction features: custom uploaded park maps with editable restaurant pins, stored latitude/longitude coordinates, visitor phone location when permission is granted, nearby restaurant results, approximate walking distance, and a Get Directions button that opens Google Maps or Apple Maps.
  • Restaurant pages: restaurant descriptions, hours, distance context, mobile ordering status, dedicated fryer indicator, allergy accommodation notes, and photo galleries.
  • Menu item pages: item descriptions, ingredients, cross-contamination notes, meal category, gluten-free status, pricing, and multiple images.
  • Search and filters: search by park, restaurant, and menu item with filters for breakfast, lunch, dinner, snacks, dedicated fryer, mobile ordering, gluten-free only, and location.
  • Reviewer verification: onboarding questions provided by Anna, with admin review before users can review restaurants or upload images.
  • Saved activity: users can save favorite restaurants, save favorite menu items, and quickly return to recently viewed content.
  • Reviews and photo uploads: approved contributors can leave ratings, write reviews, upload images, and share gluten-free/celiac context.
  • Community updates: users can submit restaurant updates, suggest menu changes, report unavailable menu items, upload photos, and share allergy accommodation experiences for admin approval.
  • Notification email configuration: setup for automated account emails, including verification and password reset messages.
  • Mobile-first responsive design: polished layouts for guests using phone, tablet, and desktop browsers while navigating the Disneyland Resort.
  • Artwork and icons: custom visual assets and interface polish for the production-ready website.
  • Launch setup: GitHub setup, domain setup through GoDaddy or Cloudflare, hosting setup, and production launch.

Private Owner Tools

Standard Build Admin Management Tools

  • Custom Content Management System (CMS): complete control over parks, maps, restaurant pins, menu items, reviews, contributor approvals, permissions, and community updates.
  • Purpose-built admin portal: a custom private owner area built specifically for the website, rather than a generic template.
  • Park Management: create, edit, delete, reorder, and upload park maps.
  • Map Management: place, move, edit, and remove restaurant pins visually on uploaded maps.
  • Restaurant Management: create, edit, archive, and manage restaurant information.
  • Menu Management: create, edit, archive, and organize menu items by restaurant and meal type.
  • Review Management: view, approve, remove, or moderate reviews and uploaded photos.
  • Reviewer Verification Management: review answers to Anna's onboarding questions, approve contributors, deny contributors, and revoke contributor permissions.
  • Permissions Management: manage which users can access admin management tools, paid content, review posting, image uploads, and contributor-only features.
  • Community Update Management: approve or reject suggested changes before they appear publicly.
  • Account Email Settings: configure the email address used for account verification and forgotten password messages.
