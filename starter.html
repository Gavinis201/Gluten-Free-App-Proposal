Project Overview

The Lean Build package is designed to launch the core informational website experience at the lowest project price. Included Features

What Is Included

  • User authentication: account creation, secure sign-in, email verification, and forgotten password reset.
  • User settings page: users can change their password or delete their account.
  • Lean Build permissions: two access levels: admin and user.
  • Payment access: subscription or one-time payment processing through Stripe or PayPal.
  • Interactive Disneyland map: map-based browsing for restaurants, food carts, snack stands, and quick-service locations offering gluten-free options.
  • Location filters: users can filter by Disneyland Park, Disney California Adventure, Downtown Disney, and hotels.
  • Restaurant detail pages: restaurant description, hours, gluten-free menu items, allergy notes, prices, and dedicated fryer indicator.
  • Food item database: food item name, description, price, gluten-free status, cross-contamination notes, and dedicated fryer status.
  • Search and filters: users can search by park, restaurant, menu item, meal type, dedicated fryer, mobile ordering, and gluten-free status.
  • Administrative content management: private admin management tools for users, permissions, parks, map locations, restaurants, menu items, descriptions, and core website content.
  • Mobile-first responsive design: layouts built for guests using phone, tablet, and desktop browsers while navigating the Disneyland Resort.
  • Lean Build artwork and icons: basic visual assets and interface polish for the first launch.
  • Notification email configuration: setup for automated account emails, including verification and password reset messages.
  • Launch setup: GitHub setup, domain setup through GoDaddy or Cloudflare, hosting setup, and production launch.

Administrative Content Management

Lean Build Admin Management Tools

  • Park Management: create and edit park records for Disneyland Park, Disney California Adventure, Downtown Disney, and hotels.
  • Map Location Management: create and edit restaurant, food cart, snack stand, and quick-service map locations.
  • Restaurant Management: create, edit, archive, and manage restaurant information.
  • Menu Management: create, edit, archive, and organize menu items by restaurant and meal type.
  • User Access Management: manage users and assign either admin or user access.
  • Account Email Settings: configure the sending email used for account verification and password reset emails.
